Сюда кладем картинки для портфолио в виде 2-х файлов для каждого портфолио: маленького размера для предпросмотра (400х570) и большого размера для полного просмотра.
Маленькие фото называем "Portfolio<№>-preview.jpg", Большие фото - "Portfolio<№>.jpg".
Все фото прогоняем через средство уменьшения размера файла:
Tiny PNG - https://tinypng.com/
Compressor - https://compressor.io/compress
Puny PNG - http://www.punypng.com/
Compress Now - http://compressnow.com/
Kraken - https://kraken.io/web-interface
Optimi Zilla - http://optimizilla.com/
JPEG Mini - http://www.jpegmini.com/main/shrink_photo
Imagify - https://imagify.io/optimizer/

а также для изменения габаритных размеров файла:
ReduceImages - https://www.reduceimages.com/
